# כריכות החוברת — generated covers

תיקייה זו מכילה את תמונות הכריכה שנוצרו לחוברת **"מערכת צירים — הרביע הראשון"**.

## עמוד השער המאושר

> **`05-workbook-cover-final.png` הוא עמוד השער הראשי, המאושר והסופי של החוברת.**

הוא — ורק הוא — משמש כעמוד הראשון של החוברת, לפני כל דפי העבודה. הקובץ מחובר אוטומטית כעמוד A4 הראשון בתצוגת "החוברת המלאה" ובהדפסה/שמירה ל־PDF (ראו `src/data/cover.ts` ו־`src/views/coverSheet.ts`).

## חלופות בלבד

`01-…`, `02-…`, `03-…`, `04-…` נשמרות **כחלופות וכחומר עיצובי עתידי בלבד**. אין להשתמש באף אחת מהן כעמוד השער הראשי.

## כיצד לעדכן את הכריכה

1. יש למקם את חמש תמונות ה־PNG בתיקייה זו (`public/assets/generated-covers/`). התיקייה מוגשת בכתובת `assets/generated-covers/` — בדיוק הנתיב המבוקש.
2. שם הקובץ המאושר חייב להישאר `05-workbook-cover-final.png`, או לעדכן את `APPROVED_COVER` ב־`src/data/cover.ts`.
3. יש לשמור על יחס A4 אנכי בתמונה. התצוגה משתמשת ב־`object-fit: contain` — התמונה אינה נמתחת ואינה נחתכת.

## סטטוס

עד שהתמונות יועלו לתיקייה זו, האפליקציה מציגה במקום הכריכה שלד־מקום (placeholder) מסומן בבירור, והבנייה נשארת תקינה.
