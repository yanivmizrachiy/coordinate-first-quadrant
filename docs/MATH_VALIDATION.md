# אימות מתמטי

כל האינוריאנטות של מערכת הצירים ברביע הראשון מרוכזות ב־`src/lib/coordinateMath.ts` ונבדקות ב־`tests/coordinateMath.test.ts`. נתוני המשחקים נבדקים ב־`tests/games.test.ts`, ותוכן החוברת ב־`tests/workbook.test.ts`. סה"כ **32 בדיקות עוברות**.

## אינוריאנטות שנבדקות

- x הוא הערך הראשון בזוג הסדור; y הוא השני.
- ציר x אופקי; ציר y אנכי (משתקף ב־`move`, `segmentParallelToAxis`).
- הרביע הראשון: x ≥ 0 ו־y ≥ 0; אין קואורדינטות שליליות (`isFirstQuadrant`).
- נקודה על ציר x ⇔ y = 0; על ציר y ⇔ x = 0 (`onXAxis`, `onYAxis`).
- תנועה ימינה מגדילה x; שמאלה מקטינה x; למעלה מגדילה y; למטה מקטינה y (`move`).
- כל מסלול מגיע ליעד המתוכנן (`applyPath`, נבדק גם למשחקים).
- שיעור x זהה ⇔ קו אנכי; שיעור y זהה ⇔ קו אופקי (`sameX`, `sameY`, `segmentParallelToAxis`).
- אורך צלע, היקף ושטח למלבן מיושר־צירים (`axisAlignedLength`, `rectPerimeter`, `rectArea`).
- הקודקוד הרביעי של מלבן (`missingRectCorner`).

## אימות משחקים

- כל מילת סוד/קוד מתקבלת בדיוק מהתשובות הנכונות (`solutionOf` מול הערך המצופה).
- היכן שנדרשת תשובה יחידה — יש בדיוק אחת (`suspectsOf(round)` באורך 1).
- כל נקודה במשחק ברביע הראשון ובתחום הגרף (`inRange`).
- המבוך פתיר (BFS ב־`mazeIsSolvable`).
- הציור הנסתר הוא מסלול סגור ברביע הראשון (`planIsClosedFirstQuadrant`).

## אימות תוכן החוברת (בדיקות חובה מ־CLAUDE.md)

- קיימים 46 עמודים ממוספרים 1–46 (38 דפי עבודה + 8 שעשועונים), ועוד עמוד שער שאינו נספר.
- הכותרת התחתונה הקנונית בכל עמוד.
- אין `x ציר` / `y ציר` (רק `ציר x` / `ציר y`).

- אין `מתקדמים` בהקשר תנועה (רק זזים/הזזה).
