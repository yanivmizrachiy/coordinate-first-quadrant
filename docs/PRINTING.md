# הדפסה — A4 מקצועי

## עקרונות

- `@page { size: A4; margin: 0 }` וכל `.sheet` בגודל `210mm × 297mm` מדויק עם `page-break-after`.
- בהדפסה מוסתרים כל רכיבי הממשק: הסרגל העליון, סרגלי הכלים, הניווט התחתון וכל אלמנט בעל המחלקה `no-print`.
- הגרפים הם SVG וקטורי (`shape-rendering: geometricPrecision`, `vector-effect: non-scaling-stroke`) — חדים בכל רזולוציה.
- הכותרת התחתונה הקנונית נשמרת בתחתית כל עמוד.
- אין חיתוך: כותרות נשברות בעברית מלאה (`overflow-wrap: anywhere`), והתוכן מוגדר `break-inside: avoid` בכרטיסים.

## מסלולי הדפסה

- **עמוד בודד** — במסך העמוד יש כפתור "🖨️ הדפסת העמוד"; רק העמוד המוצג נמצא ב־DOM, ולכן נדפס עמוד אחד.
- **כל 34 העמודים** — מסך "החוברת המלאה" מרנדר את כל הגיליונות ומאפשר "🖨️ הדפסת כל העמודים".
- **משחק** — לכל משחק כפתור "🖨️ הדפסה"; שדות הגילוי (מילה/קוד) מסומנים `no-print`, כך שנדפס גיליון עבודה נקי.

## שחור־לבן

מצב שחור־לבן (`body.grayscale-mode`, כפתור בסרגל העליון) עובר לגווני אפור מובחנים בניגודיות גבוהה עם `print-color-adjust: exact`, לקריאוּת מצוינת גם במדפסת לא־צבעונית וחיסכון בדיו.

## הורדת PDF

הדפסה מהדפדפן → "שמירה כ־PDF" מייצרת PDF ב־A4 מדויק. זהו המסלול האמין ביותר בכל דפדפן ומכשיר, ולכן לא נוספה תלות בספריית PDF.
