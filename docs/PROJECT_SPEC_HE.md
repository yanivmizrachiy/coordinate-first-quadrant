# מפרט הפרויקט — מערכת צירים, הרביע הראשון

## מטרה

לספק חוברת אחת, ברורה ונוחה לצפייה ולהורדה בנייד, בנושא „מערכת צירים — הרביע הראשון”.

## מבנה החוברת

1. עמודים 1–30: חוברת הליבה.
2. עמודים 31–34: יחידת המשך — זווית ישרה במערכת צירים.
3. סך הכול: 34 עמודים.

## דרישות עיצוב

- A4 אנכי בכל עמוד.
- כותרת ראשית מלאה שאינה נחתכת.
- מספר עמוד בתוך עיגול כחול, לפי שפת העיצוב של פרבולה.
- כותרת תחתונה קבועה:
  - `יניב רז - מדריך מחוזי חט"ב בעיר ירושלים`
  - `הדרכה במחוז ירושלים והעיר ירושלים - מנח"י, בהובלת איילת קריספין`
- תיבת תשובה קטנה בסמוך למערכת הצירים כאשר השאלה דורשת תשובה מילולית או זוג סדור.
- אין שדות שם ותאריך.
- אין כותרות או מילים שהן תוויות דמו.

## דרישות תוכן

- לשמור על שאלות מתמטיות ברורות ומדויקות.
- לא לכתוב „תלמיד סימן”; להשתמש בשם כגון „גיא סימן”.
- להסיר את המילים „ניתוח”, „טענה”, „אתגר” וכל ניסוח דמו דומה.
- לשמור על RTL בעברית ועל LTR במספרים ובזוגות סדורים.

## דרישות שימוש

- גלילה תקינה עד עמוד 34.
- בחירת עמוד מתוך תפריט.
- כפתורי קודם/הבא.
- תצוגה טובה במחשב ובנייד.
- הדפסה ושמירה כ־PDF.
- GitHub Pages מתוך הריפו החדש בלבד.
