# שיטת עריכה פשוטה

1. יניב שולח דרישות בעברית רגילה.
2. העריכה מתבצעת ישירות ב־GitHub בריפו `yanivmizrachiy/coordinate-first-quadrant`.
3. אין לבקש מיניב להריץ PowerShell, CMD, ZIP או סקריפט מקומי כאשר קיימת דרך ישירה ב־GitHub.
4. כל קבוצת דרישות מרוכזת לעדכון אחד ככל האפשר.
5. לפני שמירה קוראים את `USER_MEMORY.md`, `CLAUDE.md`, `docs/CURRENT_REQUIREMENTS.md` ו־`docs/DECISION_LOG.md`.
6. כל שינוי מחליף רק דרישה סותרת ושומר את יתר הדרישות.
7. מבצעים בדיקות תוכן, גלילה, הדפסה, מובייל וכותרות תחתונות.
8. מדווחים בסיום רק: מה השתנה, תוצאת הבדיקות, commit וכתובת התצוגה.
9. אסור לבצע כתיבה כלשהי ב־`yanivmizrachiy/parabula-next`.
