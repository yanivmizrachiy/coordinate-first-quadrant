# דרישות פעילות — מקור אמת

## מבנה
- חוברת אחת בת 34 עמודים.
- כותרת החוברת: `מערכת צירים - הרביע הראשון`.
- תצוגה מקדימה חיה ורציפה.
- PDF להורדה לנייד נשמר כמוצר נלווה.

## ניסוח
- חובה: `x ציר`, `y ציר`.
- אסור: `ציר x`, `ציר y`.
- משימות פתיחה הן משימות השלמה, לא פסקאות הסבר מוכנות.
- בכל השלמה חסר רכיב אחר: אות, תכונה, כיוון, מושג או שיעור.
- משתמשים בשמות תלמידים אמיתיים בדוגמאות.
