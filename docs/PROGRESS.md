# מצב התקדמות

- שיטת העריכה הישירה ב־GitHub פעילה.
- קובצי workflow זמניים הוסרו.
- שאלות נכון/לא נכון מאוזנות בזמן טעינת החוברת.
- לכל שורה נוספו אפשרויות סימון אחידות.
- ניסוח תנועת נקודות משתמש ב־„זזים” וב־„הזזה”.
- מצב שחור־לבן והדפסה בשחור־לבן נוספו.
- הבודק עודכן למבנה הנוכחי.
- בוטל הכיוון של העתקת כל 34 העמודים לצורך תיקונים קטנים.
- תיקונים קטנים מבוצעים רק באזורי הנתונים, הפונקציות או הסגנון הרלוונטיים.
- שינוי מבני רחב יתבצע רק כמשימה נפרדת ורק כאשר יש בו תועלת מוכחת.
