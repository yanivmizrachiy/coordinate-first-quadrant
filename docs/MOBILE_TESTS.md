# בדיקות מובייל

## שיטה

בדיקות תצוגה אוטומטיות עם Playwright (Chromium) על מגוון רוחבים, בבדיקת שלושה דברים מרכזיים: **אין גלילה אופקית**, הגרפים מרונדרים כ־SVG, והתוכן קריא ב־RTL. הבדיקה החוזרת נמצאת ב־`tests/e2e/smoke.spec.ts`.

## תוצאות (2026-07-20)

נמדד `scrollWidth − innerWidth` בכל מסך; הערך צריך להיות ≤ 1 (אין גלילה אופקית).

| מסך | 320px | 390px | 1280px |
|-----|-------|-------|--------|
| בית | 0 | 0 | 0 |
| תוכן העניינים | — | 0 | 0 |
| עמוד עבודה (3, 7) | — | 0 | 0 |
| מרכז המשחקים | — | 0 | 0 |
| משחק (מילת הסוד) | — | 0 | 0 |

- גרפים מרונדרים: עמוד 3 → 5 גרפים, עמוד 7 → 8 גרפים, מילת הסוד → 5 גרפים. ✓
- כותרות עבריות מוצגות מלאות, בלי חיתוך. ✓
- כפתורים בגודל נוח למגע (מינימום 44px). ✓

## דרישות שמומשו

- ללא גלילה אופקית בעמודי הממשק.
- טקסט קריא ללא זום; כפתורים גדולים ולא־חופפים.
- ניווט קבוע וברור; חזרה לבית ולתוכן העניינים מכל עמוד.
- שמירת המיקום האחרון בחוברת (`localStorage`) ופתיחה מחדש ממנו ("המשיכו מעמוד N").
- הגדלת דף למסך מלא בלי פגיעה בפריסה.
