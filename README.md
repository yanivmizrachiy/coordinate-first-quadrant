# Coordinate First Quadrant — 34-page workbook

חוברת דיגיטלית עצמאית בנושא **מערכת צירים — הרביע הראשון**, הכוללת 34 עמודים.

## מה כלול

- 30 דפי ליבה של מערכת הצירים ברביע הראשון.
- 4 דפי המשך בנושא זווית ישרה במערכת הצירים.
- תצוגה רציפה, ניווט לעמודים 1–34 ותצוגה מותאמת לנייד.
- עיצוב A4, מספר עמוד כחול וכותרת תחתונה קבועה.
- אפשרות הדפסה ושמירה כ־PDF.

## כללי תוכן שננעלו

- אין כותרות דמו כגון „שלב 3”, „שלב 4”, „ניתוח”, „טענה” או „אתגר”.
- אין שדות „שם התלמיד” או „תאריך”.
- כאשר מופיעה דמות תלמיד, משתמשים בשם אמיתי כגון גיא, אורי, נועה או דניאל.
- ליד שאלות גרפיות הדורשות תשובה כתובה מופיעה תיבת תשובה קטנה בסמוך למערכת הצירים.
- אין חיתוך של כותרות או טקסטים.

## הפרדה מוחלטת מהפרויקט המקורי

הריפו הזה נוצר כריפו חדש ועצמאי. סקריפט הפרסום עובד רק בתיקייה `coordinate-first-quadrant` ואינו מבצע כתיבה, commit, push או שינוי כלשהו בריפו `parabula-next`.

## הפעלה מקומית

פתחו את `index.html` בכרום.

## פרסום

הריצו את `publish-to-github.ps1`. הסקריפט יוצר ריפו ציבורי חדש בשם `coordinate-first-quadrant`, מעלה אליו את הקבצים ומפעיל GitHub Pages.

## תצוגה חיה קבועה

לחצו פעמיים על `התקנת-תצוגה-חיה.cmd`.

ההתקנה יוצרת בשולחן העבודה קיצור בשם **„תצוגה חיה - מערכת צירים”**. הקיצור פותח חלון Chrome נפרד בכתובת קבועה:

`http://127.0.0.1:4173/`

כל שמירה בקובצי הפרויקט גורמת לרענון אוטומטי. בנוסף, כאשר החוברת מפורסמת ב־GitHub Pages, הדף הפתוח בודק גרסה חדשה כל חמש שניות ומרענן את עצמו לאחר פרסום.

לסגירת השרת הפעילו `stop-live-preview.ps1`. לפרסום שינויים לריפו הנפרד בלבד הפעילו `publish-preview-update.ps1`.
